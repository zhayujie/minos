# MinOS
MinOS is a minimum available operating system.

## Documentation

Documentation can be found online at <https://zhayujie.com/tag/minos/>.

## Quick Start

### Run on QEMU

```bash
make qemu
```

### Debugging on QEMU

```bash
make qemu-gdb
```

### Make for D1-H board

```bash
make d1-board
```
